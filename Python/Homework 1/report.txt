1) Функция "Add to the front"
- Задается переменная item
- Проверка, пустой ли список, если да, то head и tail = item
- Элемент становится первым элементом связного списка
- Вывод head и tail
Efficiency: O(1)

2) Функция "Print elements one by one"
- Проверка, пустой ли список, если да, то возвращается None, если нет, то цикл проходится по всем элементам связного списка и записывает их в новый список
- Вывод нового списка
Efficiency: O(n)

3) Функция "Get an element by index"
- Проверка, пустой ли список, если да, то возвращается None, если нет, то проходится по всем элементам, параллельно подсчитывая их количество и сравнивая с индексом
- Когда находится номер элемента равный введеному индексу, происходит вывод элемента и цикл прерывается
- Если число индекса больше количества элементов в связном списке, то тогда функция прервется и выведется "Your index is out of range"
Efficiency: O(n)

4) Функция "Remove an element from the end" 
- Проверка, пустой ли списрк, если да, то возвращается None, если нет, то проходится по всем элементам связного списка, кроме последнего, и записывает их в новый список
- Заново формируются head и tail  с помощью нового списка
- Вывод head и tail
Efficiency: O(logn)

5) Функция "Remove an element from the front"
- Проверка, пустой ли список, если да, то возвращается None
- Иначе, с помощью замены переменных, меняю head
Efficiency: O(1)

6) Функция "Search for a value"
- Проверка, пустой ли список, если да, то возвращается "Your list is empty"
- Иначе, задается переменная answer = 'False', которая изменится на 'True', если по ходу прохождения по всем элементам, найдется нужный
- Прохождение по всем элементам, если находится нужный, то цикл прерывается и происходит вывод 'True', если нет, то после завершения цикла выводится 'False'
Efficiency: O(n)
